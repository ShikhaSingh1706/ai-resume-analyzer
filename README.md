# 🚀 AI Resume Analyzer

AI-powered Resume Analyzer built using Streamlit and Google Gemini API.

## 📌 Features

- Upload PDF Resume
- ATS Score Analysis
- Missing Skills Detection
- Resume Improvement Suggestions
- AI Resume Chat Assistant
- Download Improved Resume
- Error Handling with Gemini API

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- PyPDF2

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/ShikhaSingh1706/ai-resume-analyzer.git

</>Bash
pip install -r requirements.txt
streamlit run app.py

#Create Environmnet variable:
GOOGLE_API_KEY=your_api_key
